## ADR Process

1. Check highest ADR number: `ls docs/adr/*.md | sort -V | tail -1`
2. Create new ADR with next sequential number
3. Write ADR with Status: Proposed
4. Review with stakeholders
5. Update Status to Accepted (or Rejected)
6. Update this index
